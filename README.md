# 4536

「4536」は無料でダウンロードできる WordPress テーマ（テンプレート）です。

## テーマの詳細

-   SEO 内部対策
-   高速化（圧縮・遅延読み込み・インライン化など）
-   プラグインなしでの AMP 対応

など、4536 ではサイト運営に役立つ便利な機能を用意しています。

※詳しい情報は[https://4536.jp](https://4536.jp)をご覧ください。
