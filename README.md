# 4536

「4536」は無料でダウンロードできるWordPressテーマ（テンプレート）です。

## テーマの詳細

- SEO内部対策
- 高速化（圧縮・遅延読み込み・インライン化など）
- プラグインなしでのAMP対応

など、4536ではサイト運営に役立つ便利な機能を用意しています。

さらに詳しい情報は[https://4536.jp](https://4536.jp)をご覧ください。
